Deep Linking Guide

You can create links that automatically open this tool with specific settings. This is great for sharing encrypted messages or saving your workspace configuration.

Client-Side Only (No CORS Issues)

This tool runs entirely in your browser. You can link to it from any website without CORS errors.

JavaScript API v3.1

Access window.OmniEncoder in the browser console:

// Encode/Decode
await OmniEncoder.encode('hello', 'base64')
await OmniEncoder.decode('aGVsbG8=', 'base64')
// Chain multiple encoders
await OmniEncoder.chain('hello', ['base64', 'hex'])
// Auto-detect encoding
await OmniEncoder.detect('aGVsbG8=')
// Hash
await OmniEncoder.hash('hello', 'sha256')
// Listen for completion events
OmniEncoder.on('complete', (results) => console.log(results))
// Get available encoders
OmniEncoder.getAvailableEncoders()

How to construct a Link

https://OmniEncoder.vercel.app/?param=value&param2=value2

Parameters

text The text you want to load into the input box.
mode Set to encode or decode.
encoders Comma-separated list of tools to turn on (e.g., base64,hex).

Automated Extraction (Node.js Example)

To automate this programmatically, you can use a headless browser like Puppeteer. The tool emits an event when processing is finished.

const puppeteer = require('puppeteer');

(async () => {
  const browser = await puppeteer.launch();
  const page = await browser.newPage();
  
  // 1. Construct URL
  const text = encodeURIComponent("Hello World");
  const url = `https://OmniEncoder.vercel.app/?text=${text}&mode=encode&encoders=base64,hex`;
  
  // 2. Load and wait for completion signal
  await page.goto(url);
  await page.waitForFunction('window.omniComplete === true');
  
  // 3. Extract results
  const results = await page.evaluate(() => window.omniResults);
  console.log(JSON.stringify(results, null, 2));
  
  await browser.close();
})();
